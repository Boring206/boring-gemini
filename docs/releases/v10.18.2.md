# 🚀 Boring V10.18.2 - Vibe Coder Experience Upgrade

本次更新聚焦於優化 Vibe Coding 的開發流程，解決了 Cursor 測試發現的安全性假警報與依賴管理問題，讓 AI 輔助開發更順暢、更安全。

## 🛡️ 安全性與健康檢查 (Security & Health)
- **消除安全性假警報**：修改了健康檢查建議中的 `GOOGLE_API_KEY` 範例格式，避免觸發 Cursor 等 IDE 的硬編碼密鑰漏檢警報 (False Positive)。

## 🧠 RAG 依賴整合 (Dependency Management)
- **一鍵開啟 RAG 功能**：將 `chromadb` 與 `sentence-transformers` 納入 `[mcp]` 選項依賴中。
- **簡化安裝**：現在使用 `pip install "boring-aicoding[mcp]"` 將自動安裝向量記憶功能所需的所有組件，無需手動補齊。

## 💾 交易工具提示增強 (Transactions)
- **友善的引導訊息**：當在非 Git 倉庫執行交易操作時，提供更明確的錯誤提示，引導使用者執行 `git init` 以完成環境初始化。

## 📚 文件與版本更新 (Docs & Versioning)
- 同步更新繁體中文與英文版 `README` 文件中的安裝指引。
- 版本號升級至 `10.18.2` 以解決 PyPI 同版本號無法覆蓋的上傳衝突。
