# release/v10.18.1.md

## Release Title
🚀 **v10.18.1: Bilingual Intelligence Evolution & Global Knowledge Sharing** (雙語智能進化與全域知識共享)

---

## Release Notes

#### **🌟 版本亮點 (Highlights)**

本版本是 **MCP Intelligence Phase 2** 的正式完整版，除了引入四大自動化智能功能外，更達成了 **100% 全雙語文件同步**，讓全球開發者都能無障礙體驗 Vibe Coding 的魅力。

1.  **🧠 自動模式識別 (Auto Pattern Detection)**
    *   新增 `AutoLearner` 引擎，即便不手動呼叫 `learn`，AI 也會自動從「錯誤 -> 修復」的過程中學習您的程式碼偏好與慣例。
2.  **🌐 跨專案知識共享 (Cross-Project Knowledge)**
    *   引入 `GlobalKnowledgeStore`。現在，Boring 在專案 A 學到的技巧，會自動同步到 `~/.boring_brain/`，並在您開啟專案 B 時自動應用。
3.  **⚡ RAG 自動熱更新 (RAG Auto-Watcher)**
    *   新增程式碼感知監聽器，當您修改檔案後，RAG 索引會自動在背景進行微量增量更新，確保搜尋結果永遠最新。
4.  **📚 全面雙語化文件 (Full Bilingual Coverage)**
    *   同步完成 **繁體中文** 與 **英文** 的所有核心教程，包括：
        *   ADVANCED_TUTORIAL (進階開發者指南)
        *   PRACTICAL_DEMO (實戰展示)
        *   PROFESSIONAL_PLAYBOOK (專業實戰指南)
        *   TUTORIAL (完整教程)

---

#### **🛠️ 改進與修復 (Fixes & Improvements)**

*   **PyPI 發佈優化**: 修正 `v10.18.0` 發佈時的雜湊衝突，升級至 `v10.18.1` 以確保安裝路徑順暢。
*   **智能建議增強**: `boring_suggest_next` 現在更具「上下文感知」，會主動讀取 Git Diff 與 `task.md` 進度給出具體建議。
*   **CI 品質門檻**: 通過 2000+ 項單元測試與 Ruff 格式檢查，修正多項潛在的 Lint 異常。
*   **文件連結同步**: 修正所有繁中/英文文件之間的交叉連結，確保文檔索引（Docs Index）導航精確。

---

#### **⚙️ 如何更新**

如果您使用 Smithery 安裝 (推薦)：
```bash
npx -y @smithery/cli@latest install boring/boring --client gemini-cli
```

如果您使用本地 pip 安裝：
```bash
pip install --upgrade boring-aicoding
```
