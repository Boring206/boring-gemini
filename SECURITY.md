# Security Policy

## Supported Versions

The following versions of Boring are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 15.x    | ✅ Yes     |
| 14.x    | ❌ No      |
| < 14.0  | ❌ No      |

## Reporting a Vulnerability

Please report security vulnerabilities to [C112156246@nkust.edu.tw](mailto:C112156246@nkust.edu.tw).
