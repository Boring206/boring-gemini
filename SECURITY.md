## Reporting a Vulnerability

We take the security of Boring-Gemini very seriously. If you discover a security vulnerability, please follow these steps to report it:

1.  **Do NOT open a public issue.**
2.  Email the security team at `security@boring-gemini.com` (or the maintainer's email).
3.  Include a detailed description of the vulnerability, steps to reproduce, and potential impact.
4.  We will verify the issue and respond within 48 hours.
5.  Once resolved, we will publish a security advisory and credit you for the discovery.

Thank you for helping keep Boring-Gemini safe!
