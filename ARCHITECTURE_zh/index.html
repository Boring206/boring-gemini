快速入門

# 🎯 萬用入口 (推薦使用)
boring("幫我審查這段程式碼")  → 自動路由到合適工具

# 📋 常用工具
boring_rag_search     # 語義搜尋程式碼
boring_vibe_check     # 健康度評分 (0-100)
boring_session_start  # 啟動 Vibe Session
boring_commit         # 智能提交
