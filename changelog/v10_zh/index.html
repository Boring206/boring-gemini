遷移指南

從 V9.x 到 V10.x

  1. 更新安裝： 

    pip install -U boring-aicoding

  2. 新的可選依賴： 

    # 用於 RAG 功能
pip install "boring-aicoding[mcp]"

  3. 影子模式現在是預設：

  4. 設定 SHADOW_MODE_LEVEL=DISABLED 來選擇退出

  5. 建議：保持 ENABLED 以確保安全

  6. 新目錄結構： 

    ~/.boring_brain/     # 全域知識
.boring_memory/      # 專案記憶
.boring_cache/       # 驗證快取
